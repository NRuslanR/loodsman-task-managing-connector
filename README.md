# loodsman-task-managing-connector
Asynchronous integration pipeline's connector for Loodsman Tasks Subsystem. Connector consist of three parts. First of them is a request poller designed in order to receive requests from different intermediates such as message brokers like RabbitMQ. Second part is request processors that turns request into objects suitable to deal with destination system - Loodsman Tasks Subsystem. Also the request processors deals with destination system changing it state via API. Final connector consist's part is reply notificator that is reponsible for the notification sending about the request processing status into intermediates to async pipeline's source system connector.
